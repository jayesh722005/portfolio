# 🚀 Portfolio Website

A modern, responsive, and interactive personal portfolio website built with **React** and **Vite** to showcase my projects, technical skills, certifications, and experience as a MERN Stack Developer.

## 🌐 Live Demo

🔗 **Website:** https://portfolio-blue-theta-89.vercel.app/

---

## ✨ Features

- 💻 Modern and Responsive UI
- 🌙 Dark Theme
- 🚀 Fast Performance with Vite
- 👨‍💻 About Me Section
- 🛠️ Skills Showcase
- 📂 Projects Section
- 📜 Resume Download
- 📞 Contact Form
- 📱 Mobile Friendly Design

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3

### Tools
- Git
- GitHub
- Vercel
- VS Code

## 📦 Installation

```bash
git clone https://github.com/jayesh722005/portfolio.git

cd portfolio

cd frontend

npm install

npm run dev
```
